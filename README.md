# Number Guessing Game

A simple Python game where the user tries to guess a randomly generated number. The game gives hints if the guess is too high or too low and keeps track of the number of attempts.

## Features
- Random number generated every time the game runs
- User-friendly hints: "Too High" or "Too Low"
- Tracks the total number of attempts
- Works for numbers between 1 and 10 (can be easily modified)

## Requirements
- Python 3.x
